# Phishing Email Detection & Awareness Report

## Introduction

Phishing is a cyberattack in which attackers send fraudulent emails to trick users into revealing sensitive information such as passwords, banking credentials, or OTPs.

## Objective

- Analyze phishing email samples
- Identify phishing indicators
- Classify email risks
- Create user awareness guidelines

## Tools Used

- Kali Linux
- Nano Text Editor
- Git & GitHub
- Web Browser

## Common Phishing Indicators

- Generic greetings
- Fear or urgency tactics
- Suspicious links
- Unknown sender addresses
- Requests for sensitive information

## Prevention Guidelines

### Do's

- Verify sender email addresses
- Check URLs before clicking
- Enable Multi-Factor Authentication
- Report suspicious emails

### Don'ts

- Share passwords or OTPs
- Open unknown attachments
- Click suspicious links
- Respond to unverified emails

## Conclusion

Phishing attacks exploit human trust through deceptive emails. User awareness and careful verification are essential to prevent successful phishing attacks.
