# Phishing Email Detection & Awareness System

## Project Overview

This project was completed as part of the Future Interns Cyber Security Internship Program. The objective was to analyze phishing email samples, identify common phishing indicators, classify email risks, and create cybersecurity awareness guidelines to help users recognize and avoid phishing attacks.

## Objective

* Analyze phishing email samples.
* Identify phishing indicators and warning signs.
* Classify email risk levels.
* Document findings in a structured manner.
* Create awareness and prevention guidelines for users.

## Tools Used

* Kali Linux
* Nano Text Editor
* Git & GitHub
* Web Browser

## Methodology

### 1. Collection of Email Samples

Three phishing email samples were collected for analysis.

### 2. Email Analysis

Each email was examined for:

* Sender authenticity
* Generic greetings
* Urgent or fear-based language
* Suspicious URLs
* Requests for sensitive information

### 3. Risk Classification

Based on the identified indicators, each email was classified according to its risk level.

### 4. Documentation

The findings were documented along with explanations and cybersecurity awareness recommendations.

## Common Phishing Indicators

* Generic greetings such as "Dear User" or "Dear Customer"
* Suspicious or unknown sender domains
* Urgent or threatening language
* Requests for passwords, OTPs, or sensitive information
* Suspicious links that redirect to untrusted websites

## Prevention Guidelines

### Do's

* Verify sender email addresses.
* Check links before clicking.
* Enable Multi-Factor Authentication (MFA).
* Report suspicious emails to the security team.

### Don'ts

* Do not share passwords or OTPs.
* Do not click suspicious links.
* Do not download unknown attachments.
* Do not respond to unverified emails.
